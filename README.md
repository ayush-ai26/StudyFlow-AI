# StudyFlow AI

AI-powered student productivity and study planner mobile application built using React Native, Expo, and modern AI-assisted productivity tools.

---

## Features

- AI-generated study schedules
- Daily task management
- Assignment and exam tracker
- Pomodoro focus timer
- AI study chatbot assistant
- Calendar integration
- Progress analytics dashboard
- Push notification reminders
- Google sign-in and guest mode
- Modern dark/light theme
- Smooth mobile-friendly UI

---

## Tech Stack

- React Native
- Expo
- TypeScript
- Python Backend
- REST APIs
